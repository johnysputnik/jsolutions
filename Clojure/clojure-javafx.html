Introduction

In a previous introduction to JavaFX we used Java to create a small application, using an MVVM pattern. The original article can be found here.

I thought it would be interesting to find out how to do it using Clojure.

Leiningen Project File

The Leiningen project file defines the dependencies: 

1: (defproject app "0.1.0-SNAPSHOT"
2:   :description "FIXME: write description"
3:   :url "http://example.com/FIXME"
4:   :license {:name "Eclipse Public License"
5:             :url "http://www.eclipse.org/legal/epl-v10.html"}
6:   :dependencies [[org.clojure/clojure "1.7.0"]
7:                  [com.oracle/javafx-runtime "2.2.0"]]
8:   :aot :all
9:   :main app.core)

The only dependency, other than clojure is the javafx runtime.

The FXML source

The application uses an FXML file to define the structure of the UI. This should be placed in the resources folder: 

10: <?xml version="1.0" encoding="UTF-8"?>
11: 
12:  <?import javafx.scene.control.TextField?>
13:  <?import javafx.scene.layout.*?>
14: 
15:  <?import javafx.scene.control.Button?>
16:  <?import javafx.scene.control.Label?>
17:  <GridPane xmlns:fx="http://javafx.com/fxml/1" xmlns="http://javafx.com/javafx/8"
18:            alignment="center" hgap="10" vgap="10" fx:controller="app.view.Main">
19:     <children>
20:        <TextField GridPane.rowIndex="0" fx:id="textField" />
21:        <Label GridPane.rowIndex="1" fx:id="echoTextField"/>
22:        <Button GridPane.rowIndex="2" fx:id="clearButton">Clear</Button>
23:     </children>
24:     <columnConstraints>
25:        <ColumnConstraints />
26:     </columnConstraints>
27:     <rowConstraints>
28:        <RowConstraints />
29:        <RowConstraints />
30:    </rowConstraints>
31:  </GridPane>

Some JavaFX macros

Some JavaFX macros are defined and placed in src/app: 

32: ;; Some java fx macros
33: ;;
34: (ns app.jfx
35:   (:import (javafx.beans.value ChangeListener)
36:            (javafx.event ActionEvent EventHandler)
37:            (javafx.scene Scene)
38:            (javafx.fxml FXMLLoader)))
39: 
40: (defmacro fx-load-fxml [path]
41:   `(let [loc# (clojure.java.io/resource ~path)]
42:      (FXMLLoader/load loc#)))
43: 
44: (defmacro fx-add-text-listener [text-property action]
45:   `(.addListener ~text-property
46:                  (reify ChangeListener
47:                    (changed [this obsval oldval newval]
48:                      ~action))))
49: 
50: (defmacro fx-window [stage view title]
51:   `(let [root# (.loadFxml ~view)]
52:      (.setScene ~stage (Scene. root#))
53:      (.setTitle ~stage ~title)
54:      (.show ~stage)))
55: 
56: (defmacro fx-action [node action]
57:   `(.setOnAction ~node
58:                  (proxy [EventHandler] []
59:                    (handle [^ActionEvent event#]
60:                      ~action))))

The controller class

The controller class is the code behind the FXML UI: 

61: (ns app.view
62:   (:gen-class
63:    :name app.view.Main
64:    :prefix view-
65:    :methods [[loadFxml [] javafx.scene.Parent]])
66:   (:import (javafx.scene Parent))
67:   (:use [app.jfx]))
68: 
69: (defn view-loadFxml [this]
70:   "Load the FXML and create a view for display in a parent"
71:   (let [root (fx-load-fxml "window.fxml")x
72:         ;; get references to the ids in the FXML
73:         clear-button (.lookup root "#clearButton")
74:         echo-text (.lookup root "#echoTextField")
75:         edit-text (.lookup root "#textField")]
76: 
77:     ;; add an action to the clear button
78: 
79:     (fx-action clear-button
80:                (do
81:                  (.setText echo-text "")
82:                  (.setText edit-text "")))
83: 
84:     ;; add a listener to the text property on the edit control
85: 
86:     (fx-add-text-listener (.textProperty edit-text)
87:                           (.setText echo-text
88:                                     (.getText edit-text)))
89: 
90:     ;; return the root
91: 
92:     root))

The main application class

The main class initialises the application and executes it: 

 93: ;; The javafx application with a main function and an
 94: ;; overridden start method
 95: ;;
 96: (ns app.core
 97:   (:gen-class :extends javafx.application.Application)
 98:   (:import (javafx.application Application)
 99:            (javafx.scene Scene))
100:   (:use [app.jfx]
101:         [app.view]))
102: 
103: (defn -main [& args]
104:   (Application/launch app.core args))
105: 
106: (defn -start [this stage]
107:   (fx-window stage (app.view.Main.) "JavaFXML with Clojure"))
